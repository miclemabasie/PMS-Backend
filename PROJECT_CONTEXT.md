# Project: PMS (Property Management System)

## Goal
A platform for landlords and tenants to manage rent, payments, and communication in Cameroon.

## Core Features
- Tenant management
- Rent tracking
- Mobile money payments (MTN MoMo, Orange Money)
- Notifications (WhatsApp/SMS)
- Home rental Portal

## Tech Stack
- Django / docker / nginx / DRF / Djoser / Simplejwt
- Database: PostgreSQL

## Current State
- Auth system done
- Working on payments and agreements management

## Constraints
- Users are not very tech-savvy
- Must work well with mobile money
- Low bandwidth environments
